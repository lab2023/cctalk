# ccTalk

Ruby ccTalk wrapper

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'cctalk'
```

And then execute:

```bash
$ bundle install
```

Or install it yourself as:

```bash
$ gem install cctalk
```
